### 关于axios在node中的post使用
#### 环境依赖：
- node v4.4.4+ 
- npm v3.10.6+ 
- babel v6.14.0+
#### 构建：
```
git clone https://github.com/chux0519/axios-request.git
npm install
```
#### 开启服务端：
```javascript
npm run server
```
#### post请求
```javascript
npm run post
```
